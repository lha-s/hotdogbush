# Game assets

All art here is **original**, created for HotDogBush and MIT-licensed with the repo. None of it is
copied from any existing Hot Dog Bush game (the WinForms or Construct 2 versions). It is safe to
edit, replace, or redraw freely.

Swapping art: keep the same filename (or update the path in `src/game/assets.ts`). SVGs scale
cleanly; you can also drop in PNG/WebP — just change the extension in the manifest.

| File | Used for | viewBox |
|------|----------|---------|
| `stand-bg.svg` | Stage backdrop (awning, lights, counter) | 800×500 |
| `sausage-raw.svg` | Undercooked dog on the grill | 120×44 |
| `sausage-cooked.svg` | Cooked dog (servable) | 120×44 |
| `sausage-burnt.svg` | Burnt dog (toss it) | 120×44 |
| `bun.svg` | Bun under a served/plated dog | 128×56 |
| `customer-1.svg`, `customer-2.svg` | Waiting customers | 96×96 |
| `condiments.svg` | Ketchup + mustard decor | 80×60 |
